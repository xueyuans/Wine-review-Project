# Wine-review-Project
This is a data mining project for kaggle data: https://www.kaggle.com/zynicide/wine-reviews

### To run swagger:
There are three prediction swaggers:

1. Predict quality by description
Run wine_classification_description_api.py

2. Predict quality by country, province, price, taster_name using four kinds of models
Run wine_classification_api.py

3. 2. Predict quality by country, description, price, taster_name using four kinds of models
Run wine_classification_description2_api.py
